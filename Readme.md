# AquariumProject

A 3D graphics project using OpenGL with dependencies including GLFW, GLM, Assimp, and stb.

## How to start

```bash
mkdir build
cd build
cmake ..
make
 ./AquariumProject
```
## how to restart 
```bash
rm -rf *
mkdir build
cd build
cmake ..
make
```
